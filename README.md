
# Astro-Photo

Simple and scrapy static photo app built in Astro.
JSX-like syntax, compiles at runtime.
Safe for No-JS browsers.


## Command Line

Install:    `yarn install`
Start Dev:  `yarn dev`
Build:      `yarn build`
Preview:    `yarn preview`
